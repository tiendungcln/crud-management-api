# Node.js API with TypeScript

Đây là một dự án backend sử dụng Node.js, TypeScript và PostgreSQL. API hỗ trợ các chức năng cơ bản như tạo, đọc, cập nhật, xóa dữ liệu (CRUD).

## 🛠 Công nghệ sử dụng

- Node.js
- TypeScript
- Express
- PostgreSQL
- TypeORM
- dotenv

## 🚀 Cài đặt

```bash
# Cài dependencies
npm install

# Tạo file .env (xem ví dụ bên dưới)
# Chạy server dev
npm run dev
